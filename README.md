Данный веб сервис на основе ежедневных данных о курсах валют ЦБ (https://www.cbr-xml-daily.ru/daily_json.js) с использованием ASP.Net Core, реализует 2 метода:
Index — должен возвращать список курсов валют с возможностью пагинации
Currency — должен возвращать курс валюты для переданного идентификатора валюты
